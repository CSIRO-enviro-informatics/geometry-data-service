Geometry Data Service API

About

The Geometry Data Service API provides resources for retrieving geometries adhering to Linked Data principles, e.g. HTTP, Content Negotiation and URIs for resources. Some approches used in the Geometry Data Service API:

  • Store (geo-referenced) geometries (scoped to the Earth)
  • Mint URIs for geometries and provide views for it (RESTful)
  • Use well known and well supported views/semantics to describe/serve geometries, e.g. GeoSPARQL, GeoJSON, WKT...
  • Content negotiation by profiles

More details about the Geometry Data Service can be found at https://github.com/CSIRO-enviro-informatics/geometry-data-service
